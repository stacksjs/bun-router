About Bun Router

Bun Router is a high-performance, feature-rich routing library for Bun applications. It provides a robust set of tools for building web applications with Bun's native runtime.

Our goal is to provide the most efficient and developer-friendly routing solution for the Bun ecosystem, combining the best features of modern frameworks with the performance benefits of Bun.
